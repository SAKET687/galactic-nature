This FTD font galactic-nature can be used for your ftd web projects.

You can use this galactic-nature inside the FTD project. Follow below instructions to use this font package.

How to use galactic-nature?

To use this galactic-nature inside your ftd web project, if you are using our doc-site theme then you just have to pass types to ds.page

Let's say you are using doc-site inside your project then, it's as simple as making a sandwich.
Add this to your FASTN.ftd
-- fastn.dependency: saket687.github.io/galactic-nature
Lang:
ftd

Next Step

Now import galactic-nature inside .ftd file as shown below:
-- import: saket687.github.io/galactic-nature
Lang:
ftd

Next Step

add galactic-nature types to ds.page types attribute as shown below:
-- ds.page:
types: $galactic-nature.types
Lang:
ftd

Next Step

Once you add types: $galactic-nature.types inside your .ftd file, galactic-nature will be added as fastn default typography to your ds.page and to all components you add inside ds.page.

If you're not using doc-site theme and building your own fastn web theme. Then, add galactic-nature dependency into FASTN.ftd file.
-- fastn.dependency: saket687.github.io/galactic-nature
Lang:
ftd

Next Step

Once you add above dependency, then import galactic-nature inside your .ftd file:
-- import: saket687.github.io/galactic-nature
Lang:
ftd

Next Step

Done! galactic-nature is now added. Lets define a page component:
-- import: saket687.github.io/galactic-nature

-- component page:
caption title:
body body:
children wrap:
ftd.type-data types: $galactic-nature.types

-- ftd.column:
width: fill-container
spacing.fixed.px: 32
padding.px: 30
background.solid: $inherited.colors.background.base
border-width.px: 1
border-color: $inherited.colors.border-strong
border-radius.px: 10
margin-vertical.px: 10

-- ftd.text: $page.title
role: $inherited.types.heading-large
color: $inherited.colors.text-strong

-- ftd.text: 
role: $inherited.types.copy-regular
color: $inherited.colors.text

$page.body

-- ftd.column:
width: fill-container
children: $page.wrap

-- end: ftd.column

-- end: ftd.column

-- end: page
my-ds.ftd
Lang:
ftd

Next Step

In above code snippet we defined page component which has types, title, body and wrap attributes.

The types attribute used to pass a typography to it. In above example we have added $galactic-nature.types as default ftd.type-data. The ftd.type-data contains all fastn default inherited types.

The title attribute will be used for adding page titles.

The body attribute is used to add body to page component, the body can contain multiple pagragraphs, lists and any markdown format content.

The wrap attribute allows adding child components to page component. You can add any other component inside page.

Click on download icon on above code snippet my-ds.ftd file will be downloaded.

Next Step

Click on download icon on above code snippet types.ftd file will be downloaded, add previously downloaded my-ds.ftd and types.ftd inside fastn package. Update <your-fastn-package-name> with your fastn.package name and run fastn serve, load localhost web page on browser.

e.g. http://127.0.0.1:8001/types You will see the page component using galactic-nature types applied to inherited types.
